Freetts Speech Synthesizer

Freetts is an open source speech synthesizer written in Java. This can be used as a synthesizer, i.e. a way to get spoken responses, from speech recognition applications.

You need to have freetts.jar in the CLASSPATH. For more details on configuring freetts, please see using Freetts.
