Generated Sentences

This list shows all the sentences that are created using synonyms found in the synonyms table.

If you want to have new alternate sentences you can either modify the synonyms table, or right-click on the list to add alternate sentences. You can also edit any previously added sentences by clicking on it. For adding a completely different sentences, you should modify the dialog tree.
